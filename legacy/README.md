# Legacy Spark Dashboard assets

This folder contains the legacy v1 implementation of Spark Dashboard.

Contents:
- `dockerfiles_v1/`: legacy v1 container image assets based on InfluxDB and Grafana.
- `charts_v1/`: legacy v1 Helm chart for Kubernetes deployments.

For the current implementation, use the v2 assets in the repository root:
- `dockerfiles_v2/`
- `charts_v2/`

See the main repository guide for current deployment instructions.
